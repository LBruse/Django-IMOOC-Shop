# Django实现慕课网购物结算及后台管理
实现功能  
>用户登录  
>添加购物车  
>结算   
    
   
开发环境   
>Python 2.7   
>Django 1.9.8    
>xadmin 0.6.1   
   
概括  
前端直接在之前Java Web 慕课网项目的页面上改，后台完全是Python+Django实现，加上xadmin作为后台管理系统。
   
   -------------
     
总结    
在边看边敲慕课网的强力Django实战2遍之后，独自重写以前做过的项目，巩固课程内容同时也发现课程中没有提及的内容，对Django有了更进一步的认知。
